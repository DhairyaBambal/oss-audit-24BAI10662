## Project Description
This project is an open source audit of Git. It includes a short report on the origin, license, ethics, Linux footprint, ecosystem, and comparison of Git with proprietary software. It also includes five shell scripts demonstrating Linux command-line and shell scripting concepts.

## Files Included
- `script1.sh` - System Identity Report
- `script2.sh` - FOSS Package Inspector
- `script3.sh` - Disk and Permission Auditor
- `script4.sh` - Log File Analyzer
- `script5.sh` - Open Source Manifesto Generator
- `manifesto.txt` - Output generated by script 5
